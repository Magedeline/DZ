HOW TO USE THE PROJECT
======================
1. Install FMOD Studio 2.02.x or compatible
2. Open: File > Open Project > Select Pusheen.fsproj
3. Make changes to events, buses, effects, etc.
4. Build banks: File > Build Banks
5. Select platforms and click Build
6. Banks are automatically written to Audio/ folder
7. guids.txt is auto-generated or can be regenerated

IMPORTANT NOTES
===============
- Do NOT rename the project path unless updating relative paths in Workspace.xml
- Do NOT move banks manually - they are generated by FMOD Studio build process
- Mixer buses are critical for audio integration - see AudioBusManager.cs
- All bus names must match the C# code exactly for proper routing
- Events must be exported in banks for runtime access

TROUBLESHOOTING
===============
If banks don't build:
1. Check Workspace.xml output directory path
2. Verify disk space
3. Ensure no locked files in Audio/ folder
4. Try cleaning cache: Tools > Rebuild All Banks

If events not found at runtime:
1. Verify event is in the correct bank
2. Check guids.txt for correct mapping
3. Ensure event folder/names match FMOD Studio exactly
4. Banks must be loaded before playing events
