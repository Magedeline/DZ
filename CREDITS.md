# Desolo Zantas Credits And Attribution Guide

This document is a living credit description for Desolo Zantas. It is meant to help the project respect original work, inspirations, helpers, remixers, mod developers, playtesters, and the wider Celeste mod community.

## Credit Principles

- Credit every known person, team, tool, helper mod, library, remix, arrangement, reference, and inspiration source.
- Separate **original work made for Desolo Zantas** from **inspirational material**, **homage/crossover references**, **remixes/arrangements**, and **third-party helper/library work**.
- Prefer exact role names: mapping, code, sprite art, portrait art, tileset art, music composition, music remix/arrangement, FMOD implementation, sound design, writing, playtesting, accessibility feedback, bug reports, and release support.
- Ask contributors how they want to be named before release.
- Do not use “no credit” placeholders in final releases. If a source is unknown, mark it as `Source needed` and resolve it before publishing.
