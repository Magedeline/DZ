# Darker Dark Matter Source Sheets

Put the two raw sheets here with these names:

- `phase1_eye_sheet.png`
- `phase2_swordsman_sheet.png`

Then run:

```powershell
powershell -ExecutionPolicy Bypass -File .\Tools\PixelLab\import_darker_dark_matter.ps1
```

The cleaned exports go to `Graphics/Atlases/Gameplay/characters/darkerdarkmatter_pixel_lab/`.
